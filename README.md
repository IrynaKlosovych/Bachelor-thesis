# Симулятор для політичного аналізу та прогнозу результатів виборів

**Кваліфікаційна робота бакалавра** 

**Спеціальність:** 121 «Інженерія програмного забезпечення»

## Мета і завдання

**Мета випускної кваліфікаційної роботи бакалавра:** розробити симулятор для політичного аналізу та прогнозування результатів виборів на основі *характеристик представників електорату*, *зовнішніх чинників* (що можуть описувати стан країни), а також таких елементів політичної системи як *виборча система*.

**Завданнями для реалізації мети є:**
1. Розгляд та аналіз аналогів системи, що розробляється, з метою дослідження функціоналу, переваг та недоліків.
2. Визначення архітектури системи.
3. Проєктування системи та вибір засобів реалізації.
4. Розробка алгоритмів системи.
5. Реалізація функціоналу.
6. Тестування системи на відповідність поставлених вимог і задач.

## Об'єкт і предмет дослідження
**Об'єктом дослідження** є процес створення симуляції для політичного аналізу і прогнозу результатів виборів.

**Предметом дослідження** є методи і засоби для реалізації мети роботи; розробка алгоритмів та функціоналу системи визначення ймовірних результатів виборів з урахуванням характеристик електорату, зовнішніх факторів, виборчої системи.
